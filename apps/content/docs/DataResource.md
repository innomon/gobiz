# Data Resource Context

A `DataResource` represents the underlying data for a piece of content.

## Lifecycle
A data resource can be stored in various formats and locations, defined by its `DataResourceType`.

## Relationships
- **Data Resource Type**: Defines how and where the data is stored.
- **Content**: Associated with one or more `Content` records.

## Key Fields
- **Data Resource ID**: Unique identifier for the data resource.
- **Data Resource Name**: Descriptive name for the resource.
