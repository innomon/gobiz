# Tech Data Calendar Context

`TechDataCalendar` defines the availability and operational schedule for manufacturing resources like machines, work centers, or labor groups.

## Use Cases
- **Capacity Planning**: Determining available production hours for a resource.
- **Scheduling**: Planning production runs based on resource availability.

## Key Fields
- **Calendar ID**: Unique identifier for the calendar.
- **Description**: Detailed explanation of the calendar's purpose (e.g., "Main Production Shift").
