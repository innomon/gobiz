# Product Category Context

`ProductCategory` is used to group products for various purposes, such as navigation, pricing, or promotions.

## Lifecycle
Categories can have a hierarchical structure with parent and child categories.

## Relationships
- **Category Type**: Defines the purpose of the category (e.g., Catalog Category, Tax Category).
- **Parent Category**: Optional link to a parent category for hierarchy.

## Key Fields
- **Category ID**: Unique identifier for the category.
- **Category Name**: The display name of the category.
